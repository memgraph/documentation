# Getting started with Memgraph

Get Memgraph up and running on your data by following these installation steps. 

<Steps>
### Download and install Memgraph

Memgraph offers various Docker images, but we recommend using the
`memgraph-platform` image as it includes all the tools you might need to analyze
your data, such as command-line interface [mgconsole](/getting-started/cli),
visual user interface [Memgraph Lab](/visualize-data) and a graph algorithm
library [MAGE](/advanced-algorithms). 

With the [Docker](https://docs.docker.com/engine/install/) running in the
background, run the following command in the console:
 
```console
    docker run -it -p 7687:7687 -p 7444:7444 -p 3000:3000 --name memgraph memgraph/memgraph-platform
```

Other installation options are also available.

### Connect to the database
 
You can connect to the Memgraph instance using the command-line interface
mgconsole that automatically starts running upon executing the `docker run`
command or the [Memgraph Lab web application](/data-visualization/user-guide)
available within the `memgraph-platform` Docker image by visiting
`localhost:3000`.

![](/pages/getting-started/cli-lab.png)
