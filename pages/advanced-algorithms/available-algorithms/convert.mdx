## Functions

### `str2object(string)`

A function that converts the input string into an object the string represents, using Python's `json.dumps` function.

#### Input:

- `string: string` ➡ input string that is converted.

#### Output:

- `mgp.Any` ➡ output can be of various data types, depending of the content of the string.

#### Usage:

In this case, the string will be converted to a list:
```cypher
RETURN convert.str2object("[1,2,3]") AS result;
```

```plaintext
+----------------------------+
| result                     |
+----------------------------+
| [1, 2, 3]                  |
+----------------------------+
```

In this use case, the string will be converted into a map, containing a list:


```cypher
RETURN convert.str2object('{"key": [1,2,3]}') AS result;
```

```plaintext
+----------------------------+
| result                     |
+----------------------------+
| {"key": [1,2,3]}           |
+----------------------------+
```
