# MemGQL

MemGQL is a federated GQL ([ISO/IEC 39075](https://www.iso.org/standard/76120.html)) query engine that translates GQL queries into backend-native languages and executes them across graph databases. Clients connect via the Bolt protocol (port 7688) using any Bolt-compatible driver.

The long-term vision for MemGQL is to cover all data source types — structured, semi-structured and unstructured — essentially serving as a semantic layer where agents can find any data behind a single interface.
